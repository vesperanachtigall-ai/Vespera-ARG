# Master Codex

The **Vespera Master Codex** combines all three foundational documents into a single, unified reference:

**`codex/Vespera_Master_Codex.md`**  
https://github.com/vesperanachtigall-ai/Vespera-ARG/blob/main/codex/Vespera_Master_Codex.md

This codex contains:

- Core Foundations (Part 1)  
- Advanced Systems and Deep Lore (Addendum)  
- Hybrid ARG Engine and Multiyear Design (Part 3)  

It defines the canon rules for:

- Breach physics  
- Timeline structure  
- Character behavior  
- Tone and linguistic rules  
- Multi-platform continuity  
- Seasonal escalation  
- Veil succession  
- Long-form cosmology  

All ARG content must follow the Codex unless a future season intentionally revises canon.
