# Timeline Entry Template

Use this template when adding a new event to the timeline folder.

Fields to complete:

Event title  
A short description of the event.

Date written as text  
Use year month day written in plain words.

Location  
Where the event occurred.

Event level  
Minor event, major event, or annual escalation.

Event description  
Write a clear description of what happened. Avoid interpretation.

Distortion behavior  
Describe any drift, spatial compression, visual anomalies, or audio anomalies.

Artifact list  
List any artifacts created from this event.

Veil involvement  
Describe if and how the Veil responded.

Notes  
Any relevant additional information.
