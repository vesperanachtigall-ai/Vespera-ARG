# Artifact Record Template

Use this template for documenting any image, audio, video, or text artifact added to the repository.

Fields to complete:

Artifact label  
A short plain text label.

Date written as text  
Example: twenty twenty five eleven twenty six

Location  
Write the location where the artifact originated.

Artifact type  
Image, video, audio, or transcript.

Description  
Describe what the artifact shows or contains using factual language.

Distortion notes  
Describe any anomalies such as drift, spatial effects, audio patterns, or multi feed behavior.

Source  
Explain how the artifact was recovered, for example from a camera feed or field device.

Timeline link  
Indicate which timeline entry this artifact corresponds to.

Notes  
Any additional information needed for future reference.
