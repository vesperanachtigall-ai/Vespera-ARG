# Veil Post Template

Tone:
- Clinical
- Procedural
- Factual
- Neutral
- Slightly exhausted
- Never emotional, poetic, or speculative
- Never refers to Vespera directly as a person or entity with intent

Structure:
[Veil’s Artifact] OR [Veil’s Missive]
FIELD LOG: YYYY-MM-DD / HH:MM LOCAL
Location: City, State — Specific camera or location

First paragraph:
State what was found or recorded. One to three sentences. No interpretation.

Second paragraph:
Time and drift data. Include timestamp in 24-hour format, then drift in +00:00:00 format.

Third paragraph:
Describe distortions. Use terms like:
- spatial compression
- chromatic fringing
- timestamp drift
- environmental blur
- shadow geometry
- frame corruption

Fourth paragraph:
State classification using Veil terminology:
- early post-manifestation residue
- minor breach signatures
- distributed distortion event

Final line:
— The Veil
