# Veil Log Template

This template is used for creating Veil missives and field reports.
Write in clinical, procedural tone. No emotion. No metaphor. No speculation.

Fields to complete:

Log identifier  
Use a short label such as veil log followed by a number.

Date written as text  
Example format: twenty twenty five eleven twenty seven

Time of filing  
Use local time where the Veil is located.

Location  
Where the Veil is filing from.

Event summary  
Describe what was observed using factual language only.

Observed distortions  
List drift behavior, visual anomalies, audio anomalies, or device failures.

Next actions  
Describe what the Veil will do next.

Signature  
Use: dash space The Veil
