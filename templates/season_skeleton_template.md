# Season Skeleton

Use this template to outline a full season of the ARG. A season follows the structure defined in the engine folder.

Season title  
Provide a short descriptive name for the season.

Phase one  
Describe how the Long Shadow manifests in this season.

Phase two  
Describe the Ascent and how the breaches intensify.

Phase three  
Describe the Event Horizon for the year and what changes as a result.

Key artifacts  
List artifacts planned for the season.

Key Veil logs  
List important Veil missives or field reports.

Follower based triggers  
Describe optional escalation points based on user engagement.

Cross platform notes  
Describe any expansions into other platforms.

End of season notes  
Summarize how the season ends and what changes for the next one.
