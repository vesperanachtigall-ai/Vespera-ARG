# Transcript Artifacts

This folder contains text based artifacts. These may include Veil missives, field notes, log entries, dialogue transcriptions, decoded messages, or written summaries of audio or video anomalies.

Tone requirements:

Veil tone  
- Clinical and procedural  
- Precise and minimal  
- Never emotional or poetic  

Vespera tone  
- Victorian phrasing  
- Cold and elegant  
- Contempt carried through ellipses  
- Never informal

Suggested naming style:

Write the date as year month day, followed by a label describing the type of text.  
Example written in plain text:  
20251123_veil_log_1

Each transcript must correspond to a breach or analysis event within the timeline folder.
