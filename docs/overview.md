# Vespera ARG Documentation Overview

This folder contains general documentation that supports the long term structure of the Vespera ARG. These documents do not define canon directly but explain how the project is organized, how content is managed, and how rules relate across the entire repository.

The Master Codex defines the world rules.  
The engine folder defines the structural rules.  
The docs folder explains how these systems are used in practice.

This overview file provides high level orientation for contributors working with timelines, artifacts, Veil logs, and season planning.

Main documentation files include:

overview  
This file. It explains the purpose and scope of the documentation folder.

escalation rules  
Describes how breach events escalate across seasons and how to maintain consistency.

veil behavior  
Describes the tone and operational rules for writing Veil logs and field reports.

artifact handling  
Explains how to create, format, and catalog artifacts that appear in the timeline.

The goal of the documentation folder is to maintain clarity as the project grows over time.
